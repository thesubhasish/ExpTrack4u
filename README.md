# ExpTrack4u
A Python-based CLI application that allows users to record daily expenses, view spending history, and calculate total costs using in-memory data structures and robust input validation.
#
A lightweight, command-line Python application designed to help users efficiently manage their personal daily finances. Built with core Python data structures and robust error handling, this tool provides a seamless and crash-free user experience.

## Features

* **Add Expenses:** Log daily transactions with dates, categories, descriptions, and precise amounts.
* **View History:** Display a cleanly formatted list of all recorded expenses in the current session.
* **Calculate Totals:** Automatically compute and display the aggregate amount spent.
* **Input Validation:** Built-in `try-except` blocks ensure the application handles invalid user inputs (like typing letters instead of numbers) without crashing.

## Prerequisites

* Python 3.x installed on your system.
